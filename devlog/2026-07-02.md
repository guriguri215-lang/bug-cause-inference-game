# Devlog: 2026-07-02

## Summary

Implemented the first P1b MVP scaffold as a small injected-bug checkout/pricing benchmark.

## Added

- `bug_cause_inference.p1b` package with dataset metadata, executable checkout helpers, actions, policies, evaluation, and reports.
- 20 buggy variants and 5 clean variants.
- Function-level location ranking over the initial P1b candidate universe.
- Cause posterior and fix-intent category posterior updates.
- P1b policies: `random_action`, `fixed_checklist`, `test_first`, `coverage_first`, `recent_diff_first`, `cause_only_p1a_style`, and `expected_utility_per_cost`.
- CLI commands:
  - `p1b-list-variants`
  - `p1b-report`
  - `p1b-evaluate`
- JSON and Markdown examples under `examples/p1b/`.
- Tests for P1b dataset shape, action behavior, policy execution, evaluation metrics, and CLI output.

## Scope Notes

- P1b remains a small injected-bug benchmark scaffold.
- It does not generate patches.
- It does not handle large real repositories.
- It does not create real per-variant git commits or diffs.
- It does not implement adversarial bug generation or formal game-theoretic framing.
- `inspect_recent_diff` uses synthetic metadata.
- `run_property_search` uses deterministic enumerated cases rather than randomized Hypothesis-style generation.
